## Prerequisite
* Nodejs v6+
* yarnpkg (optional)


## Getting Started
```sh
git clone https://github.com/shraddhaMistry/ecommerce-react.git

cd ecommerce-react

yarn install
    or
npm install

yarn start
    or
npm start
```

Open [http://localhost:3000](http://localhost:3000)<br>
